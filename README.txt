Description of poject

The laboratory works will be written in Java. 
A Java project consists of the following components:
1) build folder - which consists of generated sources and project's classes.
		  In "built-jar.properties" is specified the path to the project.
2) dist folder -contains the .jar file used to distribute the project.This file will run the project.
3) src folder - contains the source code of the project.
4) build.xml file- this .xml specifies the way that project is going to be built.
5) nbproject folder -the folder where NetBeans saves some configuration files, build-scripts ,etc. , 
	 	     which gives IDE the possibility to work with Java project.
6) manifest.mf - the file that contains information about the files packaged in a JAR file.
